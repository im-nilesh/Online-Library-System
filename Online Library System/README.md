# Online Library System

A React + Redux online library application built using Vite.

## Features

- Home Page
- Browse Books
- Category Filtering
- Search by Title or Author
- Book Details Page
- Add New Book
- Redux State Management
- Dynamic Routing
- 404 Page

## Technologies Used

- React
- React Router DOM
- Redux Toolkit
- React Redux
- Vite
- Tailwind CSS

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
